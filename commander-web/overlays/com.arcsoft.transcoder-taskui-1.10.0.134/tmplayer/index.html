ArcSoft TotalMedia Player

Welcome to this demo page. Please install "ArcSoft TotalMedia Player" before playing back the video content. If you choose "Or directly click here to install ArcSoft TotalMedia Player.", it is strongly recommended to run installer(TMPSetup.exe) as Administrator after saving to the local storage. You may uninstall it anytime by getting in "Programs and Features" on "Windows Control Panel" and double clicking "ArcSoft Media Player Plugin".
